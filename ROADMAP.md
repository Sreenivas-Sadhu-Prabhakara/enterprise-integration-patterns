# Roadmap

The courseware site tracks the [code course](https://github.com/Sreenivas-Sadhu-Prabhakara/camel-eip-course).
Here's what's shipped and what's next.

## ✅ Shipped — v1.0
- **22 lessons** covering **18 Enterprise Integration Patterns**, plus a Spring Boot on-ramp, testing,
  error handling, broker setup, and a capstone.
- Pillar **[guide](https://sreenivas-sadhu-prabhakara.github.io/enterprise-integration-patterns/guide.html)**,
  full **[EIP map](https://sreenivas-sadhu-prabhakara.github.io/enterprise-integration-patterns/eip-map.html)**,
  3 comparison pages, and a printable **[cheat sheet](https://sreenivas-sadhu-prabhakara.github.io/enterprise-integration-patterns/cheatsheet.html)** (+ PDF).
- Self-contained, mobile-first, accessible; Lighthouse SEO 100 on every indexable page.

## 🔜 Next — planned pattern lessons
From the EIP map "reference" set: Dynamic Router, Routing Slip, Resequencer, Scatter-Gather, Normalizer,
Claim Check, Competing Consumers, Polling Consumer, Transactional Client, Circuit Breaker, Saga, Throttler.

## 🔜 Next — site features
- A Kafka-first messaging track.
- More comparison pages (Camel vs Spring Cloud Stream; JMS vs Kafka).
- An observability walkthrough page.
- A custom domain so `robots.txt`/`sitemap.xml` sit at the domain root (better crawl discovery).

## How to help
Contributions welcome — content is CC-BY-4.0. Open an issue to propose a lesson or vote on the next pattern.
